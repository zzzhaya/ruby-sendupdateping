sendupdateping

  • github.com/zzzhaya/ruby-sendupdateping

DESCRIPTION:

Send update ping to ping servers.

FEATURES:

  • Command line tool to send update pings to servers listed in the YAML config file.

SYNOPSIS:

require ‘sendupdateping’ require ‘yaml’ opts = Hash.new(YAML.load_file(‘config.yaml’) client = Sendupdateping.new(opts) client.ping # Sending pings

REQUIREMENTS:

  • Nothing so far.

INSTALL:

  • sudo gem install sendupdateping
